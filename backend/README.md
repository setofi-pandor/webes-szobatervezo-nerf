# Backend

Ez a mappa a szakdolgozathoz tartozó backend alkalmazást tartalmazza.

A backend Node.js és Express segítségével készült. Feladata a felhasználó által feltöltött videófájl fogadása, valamint a feldolgozási folyamat elindításának előkészítése.

A fájlfeltöltés kezelésére a Multer csomag került alkalmazásra.
